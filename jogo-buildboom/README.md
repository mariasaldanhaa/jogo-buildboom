# BuildBoom! 💻🔧 (EM ANDAMENTO)

> Um jogo didático sobre montagem e compatibilidade de computadores.

## 📖 Sobre o projeto

O **BuildBoom!** é um jogo educativo desenvolvido como trabalho prático final da disciplina de Programação Orientada a Objetos.

O jogo acompanha o técnico de TI **Jorge**, conhecido como *Jorginho do TI*, em uma assistência técnica localizada no centro de Bambuí-MG. O local atende usuários leigos, gamers e pessoas que tentaram montar computadores sem possuir conhecimento técnico.

O principal objetivo do jogo é ensinar, de forma interativa e divertida, conceitos relacionados à compatibilidade entre componentes de computadores.

---

## 🎮 Como funciona

O jogo é dividido em **cinco rodadas**.

Em cada rodada:
- O jogador deve escolher **1 entre 3 componentes** do mesmo tipo;
- As escolhas incluem peças como:
  - CPU
  - Placas-mãe
  - RAM
  - Fontes
  - Gabinete.

Ao final da montagem:
- O técnico **Jorge** analisa a configuração;
- O sistema informa se o computador é compatível;
- Caso exista algum erro, o jogador recebe explicações sobre:
  - quais peças causaram conflito;
  - por que a incompatibilidade ocorreu.

Dessa forma, o jogo promove aprendizado prático sobre hardware e montagem de computadores.

## 🎯 Objetivo educacional

O projeto busca auxiliar no aprendizado de:
- Compatibilidade de hardware
- Funcionamento básico de computadores
- Relação entre peças de um PC
- Tomada de decisão em montagem de computadores

Tudo isso utilizando conceitos de:
- Programação Orientada a Objetos
- Estruturação de sistemas
- Lógica de programação
